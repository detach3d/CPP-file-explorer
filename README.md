### **C++ File Searcher**  

#### **Overview**  
This is a simple **File Searcher** written in **C++**, allowing users to search directories and files.  

#### **Features**  
- Search files and directories  

#### **Requirements**  
- C++17 or later  
- Standard C++ libraries (filesystem, exception)  
- wxWidgets
